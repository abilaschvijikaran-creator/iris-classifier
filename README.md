#Iris Classifier (Decision Tree) 

##Overview
This project builds a simple machine learning model that classifies iris flowers into three species
(*setosa*, *versicolor*, and *virginica*) using a Decision Tree algorithm from **scikit-learn**. 
It's part of the AI Fundamentals course.
